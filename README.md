# 🍕 Pizza Sales Dashboard

## Overview
This repository showcases an interactive **Power BI dashboard** built to analyse pizza sales performance. The report provides insights into sales trends, customer behavior, and product performance through two main report pages.

---

## 📊 Dashboard Structure

### 1. Pizza Sales Summary
This page highlights overall performance with key metrics:
- Daily Orders & Monthly Orders  
- % of Sales by Pizza Category  
- % of Sales by Pizza Size  
- Pizzas Sold by Category  

---

### 2. Best / Worst Sellers
This page focuses on product-level insights:
- Top 5 & Bottom 5 Pizza Categories by:
  - Revenue  
  - Quantity Sold  
  - Total Orders  

---

## 📌 Key KPIs (Available Across Both Pages)
- Total Revenue  
- Average Order Value  
- Total Pizzas Sold  
- Total Orders  
- Average Pizzas per Order  

---

## 🎯 Purpose
The dashboard helps to:
- Identify top and underperforming products  
- Understand sales distribution across categories and sizes  
- Monitor overall business performance

- Power BI

- Interactive Visualizations
